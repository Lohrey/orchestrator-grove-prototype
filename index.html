Provider: Ollama

Install + model guide

Local mode calls Ollama from your browser. Use http://127.0.0.1:11434 and allow this site in Ollama CORS.

TabbyAPI uses the local OpenAI-compatible endpoint http://127.0.0.1:5000/v1. This setup already allows permissive local CORS.

Windows Ollama setup

PowerShell:

$env:OLLAMA_HOST="127.0.0.1:11434"
$env:OLLAMA_ORIGINS="https://docs.pau1.cloud"
ollama serve

CMD:

set OLLAMA_HOST=127.0.0.1:11434
set OLLAMA_ORIGINS=https://docs.pau1.cloud
ollama serve

Wichtig auf Windows: Falls Ollama schon als App/Tray-Service läuft, kommt oft:

bind: address already in use

Dann:

  1. Ollama im Tray beenden / Quit Ollama
  2. Terminal neu öffnen
  3. PowerShell-Befehle ausführen
  4. ollama serve

Dauerhaft setzen PowerShell:

setx OLLAMA_HOST "127.0.0.1:11434"
setx OLLAMA_ORIGINS "https://docs.pau1.cloud"

Danach Ollama komplett neu starten.

Dann im Spiel: Esc → Local AI → Use my local Ollama.

Default model: gemma4:12b. Click Refresh to load provider-specific model names.
